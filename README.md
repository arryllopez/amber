# Amber

brief readme thanks claude

Look through deep time.

Amber is a paleogeographic AR app that transforms your surroundings into a scientifically accurate window into prehistoric Earth. Point your phone at the world around you and see what that exact location looked like millions of years ago — grounded in real plate tectonic reconstruction, paleoclimate data, and fossil records.

## Modes

**Field Mode** — Walk your real environment in AR. GPS anchors the experience to your exact location. As you move, the prehistoric world moves with you.

**Explorer Mode** — Browse an interactive 3D globe from anywhere. Drop into a first-person view of any location on Earth at any point in geological time.

**Group Walk** — Join a shared session with friends. Walk the same prehistoric world together, see each other as avatars in real time.

## Stack

- iOS — Swift, ARKit
- Android — Kotlin, ARCore
- Backend — Go
- Web — React

## Data Sources

- [GPlates Web Service](https://gws.gplates.org) — tectonic plate reconstruction
- [PALEOMAP Project](https://www.globalgeology.com) — paleogeographic maps
- [PaleoClim](http://www.paleoclim.org) — paleoclimate surfaces
- [Paleobiology Database](https://paleobiodb.org) — fossil occurrence records
